📋 Пояснительная записка

Макет разработан для образовательной платформы Tech Interface на основе дизайн-системы проекта: стиль neumorphism (светлые блоки с объёмными тенями) и glassmorphism (тёмные блоки с blur-эффектами). Цветовая палитра строго соответствует презентации: #1F2A44 — основной тёмно-синий фон, #3B82F6 — акцентный голубой для CTA-кнопок и иконок, #93C5FD — светло-голубой для второстепенных элементов, #E5E7EB — нейтральный серый. Использована 12-колоночная сетка с gap 24px, что обеспечивает консистентность отступов на всех блоках. Типографика: Montserrat 900 для заголовков (максимальная жирность, отрицательный letter-spacing для плотности), Inter для основного текста — та же пара, что в презентации.

Визуальная иерархия выстроена по принципу: главный заголовок Hero (64px) → заголовки секций (44px) → названия карточек (18px) → тело текста (14px). Принцип F-паттерна: взгляд проходит слева направо по хедеру, затем по Hero-блоку вниз к CTA, далее к карточкам и снова к CTA-баннеру внизу страницы — два ключевых CTA обрамляют контент сверху и снизу. Контрастность текста соответствует WCAG AA: светлый текст на тёмном фоне (4.5:1+), тёмный текст на светлом фоне. Интерактивность реализована через hover-эффекты подъёма карточек и свечения кнопок, анимации появления при загрузке страницы.
